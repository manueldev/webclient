### Swing music client with Download track buttons

This repo contains the client code for the [swing music player](https://github.com/geoffrey45/swingmusic) (but with Download track buttons).

### Setup

This project requires `Node v14` or newer and `yarn@1.22.19`. Install them first.

- [NodeJS](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com)

Clone the repository and install the dependencies.

```bash
git clone https://github.com/geoffrey45/swing-client.git

cd swing-client

# Install dependencies
yarn install

# Start the development server
yarn dev

# Build the project
yarn build
```

---
### Generating PWA favicons

```sh
yarn generate-pwa-assets
```

### Using this custom frontend with Swing Music

To use this customized frontend (with Download track buttons) together with [swingmusic](https://github.com/swingmx/swingmusic), first build the client and then point the server to the generated `dist` folder.

```bash
# 1. Install dependencies and build the client
yarn install
yarn build:force

# 2. Go to the swingmusic server repo
cd /path/to/swingmusic

# 3. Run the server pointing to the custom client's dist folder
python3 run.py --client /path/to/webclient/dist
```

Replace `/path/to/swingmusic` with the absolute path where you cloned [swingmx/swingmusic](https://github.com/swingmx/swingmusic), and `/path/to/webclient/dist` with the absolute path to the `dist` folder generated by `yarn build:force` in this repo.
